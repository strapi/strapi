# My Strapi Task 1 (Submission)

This project is the submission for **Task 1**, demonstrating the setup of a local Strapi environment and content modeling using Yarn.

## 🚀 Project Overview

- **Project Name:** my-strapi-task1
- **Goal:** Initialize Strapi, connect to GitHub, and create a custom Content Type.
- **Created With:** `yarn create strapi-app`

## 🛠 Features Implemented

1.  **Project Initialization:** Successfully created a fresh Strapi project running on port `1337`.
2.  **Content Modeling:** Created a new Collection Type named **"Article"** with the following fields:
    - `Title` (The 2026 Guide to Production-Grade Kubernetes CI/CD)
    - `Content` (Production)
3.  **Git Workflow:**
    - Initialized repository and linked to GitHub.
    - Implemented feature branching (`task-1-setup`).

## 📦 How to Run This Project

To run this project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/mohsinmir-101177/my-strapi-task1.git](https://github.com/mohsinmir-101177/my-strapi-task1.git)
    ```

2.  **Install dependencies:**

    ```bash
    cd my-strapi-task1
    yarn install
    ```

3.  **Start the development server:**

    ```bash
    yarn develop
    ```

4.  **Access the Admin Panel:**
    - Open your browser and go to: [http://localhost:1337/admin](http://localhost:1337/admin)
